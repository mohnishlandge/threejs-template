[Demo](https://threejs-template.netlify.app/)


## Installation

Install dependencies (only for first time) :

```
yarn
```

Compile the code for development and start a local server:

```
yarn dev
```

Local Server will open on:

```
http://localhost:8080/
```

Create the build:

```
yarn build
```
